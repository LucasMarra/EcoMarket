# EcoMarket - Sustainable Marketplace

A sustainable product marketplace built with React that promotes conscious consumption through sustainability badges and a points-based reward system.

## About

EcoMarket is a web application that helps users find and purchase sustainable products. Each product features sustainability indicators and certifications, and users earn points for making eco-friendly purchases.

## Features

- Browse sustainable products with detailed sustainability information
- Sustainability badges and certification system
- Shopping cart functionality
- User points and rewards system
- Product filtering by category and sustainability level
- Mock checkout process
- User profile with purchase history and impact tracking

## Technologies

- React 18
- React Router DOM
- JavaScript (no TypeScript)
- CSS3

## Getting Started

### Installation

```bash
npm install
```

### Running the Application

```bash
npm start
```

The application will open at `http://localhost:3000`
